# Lectio Next.js

Educational component library built on Next.js + TypeScript + Tailwind + shadcn/ui primitives.

## Agent Standards

Read `agents/ENTRY.md` for development standards, workflows, and quality gates.

## Quick Reference

```bash
npm run dev        # Start Next.js dev server
npm run lint       # ESLint
npm run typecheck  # TypeScript checking
npm run test       # Vitest component and schema tests
npm run build      # Production build
```
