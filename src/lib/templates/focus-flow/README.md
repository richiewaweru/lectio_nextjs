# Focus Flow

## Intent
Reduce cognitive overload and present one meaningful chunk at a time without watering down the concept.

## Best For
- ADHD-friendly support
- Dyslexia-sensitive reading flows
- Scaffolded classrooms and intervention settings

## Not Ideal For
- Dense comparison lessons
- Topics that require multiple simultaneous visuals

## Learner Fit
- ADHD-friendly
- Dyslexia-sensitive
- Scaffolded

## Subjects
- Mathematics
- Science
- English

## Lesson Flow
Hook -> Short Explain -> Support Cue -> Practice -> Reflect -> What Next

## Required Components
- SectionHeader
- HookHero
- ExplanationBlock
- PracticeStack
- WhatNextBridge

## Optional Components
- DefinitionCard
- GlossaryInline
- PitfallAlert
- ReflectionPrompt

## Why This Template Exists
Some support needs are structural, not cosmetic. This template lowers the number of competing elements and keeps the learner in one calm reading rhythm.

## Responsive Notes
The lesson keeps the same single-column pacing at every breakpoint. No sidebar appears on larger screens.

## Print Notes
The flat practice list and the chunked order stay intact in print.
