# Timeline Narrative

## Intent
Teach through chronology so the learner experiences the concept as forward motion through time.

## Best For
- History lessons and discovery stories
- Topics where sequence and cause matter more than static definition
- Learners who remember ideas as a story

## Not Ideal For
- Pure classification lessons
- Direct procedure training

## Learner Fit
- Narrative
- General

## Subjects
- History
- Science
- English

## Lesson Flow
Hook -> Timeline -> Explain -> Reflect -> Practice -> What Next

## Required Components
- SectionHeader
- HookHero
- TimelineBlock
- PracticeStack
- WhatNextBridge

## Optional Components
- ExplanationBlock
- ReflectionPrompt
- PitfallAlert

## Why This Template Exists
Some topics make the most sense when the learner feels the forward motion from one event to the next. This template makes chronology the structure.

## Responsive Notes
The timeline remains high in the reading order and degrades to a stacked event view on smaller screens.

## Print Notes
The interactive timeline becomes a vertical event list with the reflection and practice beneath it.
