# Guided Concept Compact

## Intent
Keep the guided concept arc, but compress it for confident learners and tighter teaching windows.

## Best For
- Confident learners who still benefit from a clear arc
- Shorter class segments
- Quick revision with light scaffolding

## Not Ideal For
- Learners who need heavy side support
- Figure-led or comparison-led lessons

## Learner Fit
- General
- Advanced

## Subjects
- Mathematics
- Science
- Economics
- English

## Lesson Flow
Hook -> Explain -> Define -> Example -> Practice -> What Next

## Required Components
- SectionHeader
- HookHero
- ExplanationBlock
- PracticeStack
- WhatNextBridge

## Optional Components
- DefinitionCard
- WorkedExampleCard
- PitfallAlert

## Why This Template Exists
Sometimes the full guided path is more support than the class needs. This version keeps the teaching logic while trimming the layout overhead.

## Responsive Notes
The template stays single-column and avoids sidebar behaviors entirely.

## Print Notes
Accordion content expands and the shorter arc stays intact in print.
