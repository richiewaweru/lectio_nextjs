# Formal Track

## Intent
Move quickly into formal definitions and rigorous reasoning with minimal hand-holding.

## Best For
- Advanced learners
- Proof-adjacent and definition-heavy lessons
- Print-first or exam-prep settings

## Not Ideal For
- First exposure with weak conceptual footing
- Learners who need strong visual or vocabulary support

## Learner Fit
- Advanced
- Analytical

## Subjects
- Mathematics
- Physics
- Economics

## Lesson Flow
Hook -> Formal Explanation -> Definition -> Worked Case -> Practice -> What Next

## Required Components
- SectionHeader
- HookHero
- ExplanationBlock
- DefinitionCard
- PracticeStack
- WhatNextBridge

## Optional Components
- WorkedExampleCard
- PitfallAlert

## Why This Template Exists
Some classrooms want the rigorous version right away. This template keeps the surface quiet and lets formal reasoning take the lead.

## Responsive Notes
The layout stays single-column and print-first across all breakpoints.

## Print Notes
The page is designed to read cleanly as a static worksheet-style lesson with no sidebar dependencies.
