# Process Trainer

## Intent
Teach a repeatable procedure through an ordered step sequence backed by a worked case and practice.

## Best For
- Repeatable methods in math and science
- Checklist-driven tasks
- Lessons where order is non-negotiable

## Not Ideal For
- Narrative topics
- Classification lessons with no execution order

## Learner Fit
- General
- Scaffolded

## Subjects
- Mathematics
- Chemistry
- Physics

## Lesson Flow
Hook -> Procedure -> Worked Case -> Practice -> What Next

## Required Components
- SectionHeader
- HookHero
- ProcessSteps
- PracticeStack
- WhatNextBridge

## Optional Components
- WorkedExampleCard
- PitfallAlert
- ExplanationBlock

## Why This Template Exists
Some lessons are really about executing a method correctly. This template keeps the procedure unmistakable and easy to rehearse.

## Responsive Notes
The page stays single-column so the learner can follow the method in order without scanning sideways.

## Print Notes
The full process expands in print and the practice remains directly beneath the method.
