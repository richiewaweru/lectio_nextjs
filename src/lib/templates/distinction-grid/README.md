# Distinction Grid

## Intent
Teach three or more related ideas through explicit classification criteria that stay visible while the learner applies them.

## Best For
- Civics, biology, and social science classifications
- Learners who confuse adjacent categories
- Multi-way contrasts that need a visible grid

## Not Ideal For
- Binary contrasts
- Narrative or process-led lessons

## Learner Fit
- Analytical
- Advanced

## Subjects
- Civics
- Biology
- History

## Lesson Flow
Hook -> Frame -> Classify -> Apply -> What Next

## Required Components
- SectionHeader
- HookHero
- ComparisonGrid
- PracticeStack
- WhatNextBridge

## Optional Components
- DefinitionFamily
- InsightStrip
- PitfallAlert

## Why This Template Exists
Some lessons are really classification problems. This template keeps every category and criterion visible so the learner can sort with confidence.

## Responsive Notes
The grid may scroll horizontally on smaller screens, but the classification practice stays immediately below it.

## Print Notes
The wide grid prints as a static table and the flat practice sits beneath it like a worksheet.
