# Figure First

## Intent
Anchor understanding in one dominant figure before prose explains what the learner is already seeing.

## Best For
- Biology, chemistry, geography, and anatomy lessons
- Spatial concepts that need a visible anchor
- Visual-first learners

## Not Ideal For
- Purely formal or proof-style lessons
- Topics with weak visual structure

## Learner Fit
- Visual
- General

## Subjects
- Biology
- Chemistry
- Geography
- Science

## Lesson Flow
Hook -> Diagram -> Explain -> Process -> Practice -> What Next

## Required Components
- SectionHeader
- HookHero
- DiagramBlock
- ExplanationBlock
- PracticeStack
- WhatNextBridge

## Optional Components
- GlossaryRail
- ProcessSteps
- PitfallAlert

## Why This Template Exists
Some lessons click only after the learner can point at the structure. This template gives the visual surface priority and lets the prose arrive second.

## Responsive Notes
The figure stays high in the reading order at every breakpoint and the supporting process notes stack beneath it on smaller screens.

## Print Notes
The figure becomes a full-width static print asset and the process steps expand fully.
