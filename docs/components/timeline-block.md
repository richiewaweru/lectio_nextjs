# TimelineBlock

`TimelineBlock` is the governed chronology surface for templates that teach through sequence, story, and cause over time.

## Purpose
- Turn a set of dated events into a readable instructional spine
- Support both static event stacks and the `timeline-scrubber` interaction
- Preserve a clean vertical fallback for print

## Content shape
- `title`
- `intro`
- `events`
- `closing_takeaway`

## Behaviour
- `static`
- `timeline-scrubber`

## Used by
- `timeline-narrative`
